CZ
Portfolio obsahuje 2 hry (Pong a Bird) a aplikaci na evidenci tržeb (Evidence).

Hra Pong a aplikace Evidence jsou psány v jazyce C++ a s použitím knihovny SFML. Tyto zazipované soubory stačí jen stáhnout rozzipovat a je možné je spustit.
V programu Evidence je třeba vždy nejdříve vybrat jednu z položek v seznamu nalevo a poté je možno kliknout na jedno z 6 barevných tlačítek. Heslo pro vstup do záložky pokladna je 1.

Hra Bird byla vytvořena pomocí Unity enginu a je rozdělena do dvou zip souborů. Tyto je třeba stáhnout, rozzipovat a poté složku Bird_Data vložit do složky Bird. Poté je již možné hru spustit.

EN
Portfolio contains 2 games (Pong and Bird) and application for sales records (Evidence).

Pong game and application Evidence are writen in C++ and with use of SFML library. These zipped files just needs to download, unzip and you can run them.
In Evidence program is needed to select one of the component on the left firstly, and than you can click on one of 6 colored buttons. Password for entering the pokladna is 1.

Bird game was made using Unity engine and is divided into 2 zipped files. These files you need to download, unzip and than the folder Bird_Data put into the folder Bird. After that you can run the game.
